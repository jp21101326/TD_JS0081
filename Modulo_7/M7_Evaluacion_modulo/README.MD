### `validateFields.js`
- Verifica que los campos obligatorios (como `nombre`, `correo`, `contraseГ±a`) estГ©n presentes.
- Evita peticiones incompletas y mejora la validaciГіn del lado servidor.

### `errorHandler.js`
- Maneja todos los errores de forma centralizada.
- Devuelve respuestas JSON coherentes ante errores de validaciГіn o de base de datos.

## Relaciones

RelaciГіn **muchos a muchos (N:M)** entre `Usuario` y `Rol` mediante la tabla intermedia `UsuarioRol`:

```js
Usuario.belongsToMany(Rol, { through: 'UsuarioRol', as: 'roles', foreignKey: 'usuarioId' });
Rol.belongsToMany(Usuario, { through: 'UsuarioRol', as: 'usuarios', foreignKey: 'rolId' });
```
