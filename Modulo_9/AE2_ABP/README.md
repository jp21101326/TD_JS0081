# Portafolio Web Personal -- README

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un portafolio web personal
creado durante el módulo **M9 Desarrollo de Portafolio de un Producto
Digital**. El objetivo es presentar de forma clara y profesional mi
perfil, habilidades, proyectos y medios de contacto.

El portafolio fue diseñado inicialmente con un boceto base y
posteriormente refinado mediante revisión, depuración, feedback y
ajustes finales, logrando una versión más limpia, moderna y alineada a
buenas prácticas de diseño web.

------------------------------------------------------------------------

##  Funcionalidades Implementadas con Éxito

-   Sitio completo con secciones: **Home**, **Sobre mí**,
    **Habilidades**, **Proyectos**, **Servicios** y **Contacto**.
-   Diseño responsivo básico adaptado a distintos tamaños de pantalla.
-   Galería de proyectos organizada visualmente con imágenes
    representativas.
-   Barra de navegación funcional con enlaces activos.
-   Sección de servicios con tarjetas e iconografía.
-   Estructura semántica HTML optimizada.

------------------------------------------------------------------------

## Mejoras Implementadas

Durante la etapa de depuración y refactorización se realizaron las
siguientes mejoras:

### Correcciones y ajustes

-   Limpieza y reorganización completa del código HTML y CSS.
-   Eliminación de estilos antiguos y degradados oscuros.
-   Cambio a un diseño **light/minimalista con tonos azules
    institucionales**.
-   Optimización de imágenes para una carga más rápida.
-   Ajustes en responsividad, especialmente en la sección de proyectos.
-   Etiquetas alternativas y mejoras de accesibilidad.

### Refactorización

-   Agrupación de clases CSS para evitar duplicaciones.
-   Uso de variables CSS para colores principales.
-   Renombre de clases siguiendo un patrón uniforme.
-   Reorganización de carpetas del proyecto:\
    `/css`, `/img`, `/js`, `index.html`.

------------------------------------------------------------------------

## Feedback Recibido

Se solicitaron comentarios externos y se identificaron estas mejoras:

-   Agregar más información personal en "Sobre mí".
-   Ajustar proporciones de imágenes en la sección de proyectos.
-   Modernizar las tarjetas visuales.
-   Quitar bordes visibles y mantener un estilo más minimalista.
-   Jerarquizar mejor el contenido textual.

### Cambios aplicados según feedback

-   Implementación de un rediseño completo del CSS.
-   Nuevas cards descriptivas para cada proyecto.
-   Iconos modernos en la sección de contacto.
-   Expansión del contenido personal y profesional.
-   Compresión y estandarización de imágenes.
