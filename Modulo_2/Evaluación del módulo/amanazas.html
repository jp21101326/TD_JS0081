Ataques DDoS

Sobrecarga de servidores con tráfico masivo para hacerlos inaccesibles.

Información Detallada

¿Cómo funciona? Los atacantes envían correos, mensajes o crean sitios web falsos que imitan organizaciones legítimas.

Señales de alerta:
  • Errores ortográficos y gramaticales
  • Urgencia excesiva en el mensaje
  • Enlaces sospechosos
  • Solicitudes de información personal
Prevención: Verifica siempre la fuente, no hagas clic en enlaces sospechosos y usa autenticación de dos factores.

¿Qué es? Software malicioso que cifra archivos y exige pago para su liberación.

Métodos de infección:
  • Correos electrónicos con archivos adjuntos maliciosos
  • Descargas de software pirata
  • Vulnerabilidades en sistemas desactualizados
Prevención: Mantén backups regulares, actualiza software y capacita a los usuarios.

Tipos principales:
  • Virus: Se replica y se adjunta a otros archivos
  • Gusanos: Se propaga automáticamente por redes
  • Troyanos: Se oculta como software legítimo
  • Spyware: Recopila información sin consentimiento
Protección: Usa antivirus actualizado, evita descargas sospechosas y mantén el sistema actualizado.

¿Cómo funcionan? Múltiples sistemas comprometidos envían tráfico masivo a un objetivo para sobrecargarlo.

Tipos de ataques:
  • Volumétricos: Saturan el ancho de banda
  • De protocolo: Explotan debilidades en protocolos
  • De aplicación: Atacan aplicaciones específicas
Mitigación: Usa CDN, filtrado de tráfico y servicios de protección DDoS.
